# Farmyad_Software_Engineering_PROG102_BIT1202F_Group3_ihn_name_-_dpg-.
Farmyad is a digital agriculatural market place designed to connect rural farmers directly with buyers in Freetown and other part of Sierra Leone. Throught this platform, farmers can register, advertise their available produce, set price and receive orders from buyers without them traveling long distance to sell there goods.
